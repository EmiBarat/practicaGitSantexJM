1. Para poder trabajar en el repo, primero tienen que clonarlo.
2. Hacer pull para asegurarse que tienen la versión mas actualizada del repo
3. Crear un branch con el siguiente formato: ramaEmiBarat
4. Asegurarse que estan trabajando en su branch.
5. Modificar el archivo lista.txt agregando su nombre.
6. Hacer un commit con la modificación
7. Solicitar un pullRequest a la rama principal.
